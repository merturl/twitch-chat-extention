## twitch-chat-extention

> 트위치 실시간 스트리밍 중 특정 유저의 채팅내용만 필터링 하는 크롬 익스텐션

### Project Stack

- React
- Redux
- Redux Toolkit
- TypeScript
- React icons
- React Toastify
- React scroll to bottom
- Styled Components
- Webpack
- Tmijs

## 🚀 Getting Started

# development

```
yarn start
```

# production

```
yarn start
```
